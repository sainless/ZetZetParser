Чтобы у тебя все работало корректно(и без ошибок), следуй по моим указаниям.

1) Установи необходимы зависимости, при помощи команды - pip install -r requirements.txt
2) Запусти main.py через командную строку: python3 main.py
3) Все полученные данные ты можешь сохранять в любом формате, в рамках скрипта можно: CSV либо JSON
    CSV - это удобочитаемый формат в редакторе Excel, вид приятный и красивая структура.
    JSON - это как словари(грубо говоря, объекты) ее ты можешь открыть как текстовый файл, так и с помощью программы.

4) Ждёшь несколько секунд и наслаждаешься.

P.S
